# 🤖 DS Chatbot — NLP Deep Learning

A conversational chatbot built using Deep Learning and NLP for domain-specific query handling and intent classification.

## 🚀 Features
- Intent-based response generation
- Deep Learning model for classification
- Flask web interface
- Real-time chat in browser

## 🛠️ Tech Stack
- Python
- TensorFlow / Keras
- NLP (Tokenization, Lemmatization)
- Flask
- HTML, CSS, JavaScript

## 📁 Project Structure
```
ds-chatbot/
├── app.py              # Flask web app
├── chatbot.ipynb       # Model training notebook
├── processor.py        # Prediction logic
├── chatbot_data.json   # Intents & responses
└── templates/
    └── index.html      # Chat UI
```

## ▶️ How to Run
```bash
pip install -r requirements.txt
python app.py
```
Open browser → http://localhost:5000
